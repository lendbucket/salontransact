# SalonTransact Design System

Canonical design spec. Import primitives from `components/ui/*`.

## Typography
Font: Inter, letter-spacing: -0.31px. Never font-weight 700+ except numbers >= 32px.

## Colors
Backgrounds: `#FBFBFB` page, `#FFFFFF` cards, `#F4F5F7` inputs, `#F9FAFB` hover, `#E6F4F8` active.
Text: `#1A1313` primary, `#4A4A4A` body, `#878787` muted, `#ABABAB` placeholder.
Borders: `#E8EAED` default, `#D1D5DB` inputs, `#F4F5F7` subtle.
Brand: `#017ea7` primary, `#015f80` border, `#0290be` hover, `#E6F4F8` tint.
Success: `#22c55e`/`#DCFCE7`/`#15803D`. Warning: `#F59E0B`/`#FEF3C7`. Danger: `#DC2626`/`#FEF2F2`.

## Buttons
Primary: 36px, gradient, `#015f80` border, 8px radius, inset shadow. Hover: translateY(-1px).
Secondary: 36px, white, `#D1D5DB` border. Ghost: transparent. Danger: `#FCA5A5` border.

## Cards
Shadow 2 (6-layer), 12px radius, 24px padding. Mobile: 8px radius, 16px padding.

## Status Pills
22px height, 11px/600 uppercase, 11px radius, 6px dot.

## Spacing
8px grid: 4 8 12 16 20 24 32 40 48 64 96.

## Icons
lucide-react, 16px default, stroke 1.5.
